### 1：登录协议

> 当客户端连接成功后发起的协议



请求参数

| 名称 | 类型   | 说明       |
| ---- | ------ | ---------- |
| code | int    | 固定值：1  |
| name | string | 唯一的名称 |
| msg  | string | 发送的内容 |



返回参数

| 名称       | 类型   | 说明                 |
| ---------- | ------ | -------------------- |
| returnCode | int    | 0表示失败，1表示成功 |
| msg        | string | 返回的相关说明       |
|            |        |                      |

### 3：群聊协议(client->server)

| 名称 | 类型   | 说明         |

| returnCode | int    | 0表示失败，1表示成功 |
